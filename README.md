# vidcp

Local-first CLI that ingests video files and turns them into searchable
knowledge: metadata, scenes, keyframes, transcript, OCR text, and embeddings,
stored in SQLite. CPU-only, no cloud APIs (except one-time model downloads).

**Status:** in development. See [`PLAN.md`](PLAN.md) for the implementation
roadmap. Full install/usage docs land in Step 8.
